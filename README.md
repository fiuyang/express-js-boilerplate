# Getting Start:
Before you running the program, make sure you've run this command:
```bash
$ npm install
$ cp .env.example
```

### Database setup:
- Create your own database, and put the credential in env file
```bash
# run the migration
$ npx prisma migrate dev --name init
```

### Run the program 
```bash
$ npm run start
```

### Run unit test
```bash
$ npm run test
```

### Program Will Run
```bash
$ http://localhost:3000/
```
